Proyectos

lbh-tiktok-adapter — Adaptador de Protocolo TikTok E-commerce

2026 · github.com/Thrumanshow/lbh-tiktok-adapter

Adaptador de protocolo binario para procesamiento de órdenes TikTok. Transforma eventos JSON en feromonas LBH de 16 bytes con verificación de integridad CRC-16.

  • 89.3% de reducción de ancho de banda — 150 bytes → 16 bytes
  • A 1B eventos/día: 139 GB → 14.9 GB ahorrados diariamente
  • Bloqueo SQLite basado en reclamación — cero condiciones de carrera
  • Retry + backoff exponencial + dead-letter queue
  • RFC-LBH-0004 (Integridad de Datos) + RFC-LBH-0006 (Señales)
  • Arquitectura para Salesforce Commerce Cloud, Shopify, Channel Advisor

LBH-Net — Red Swarm P2P Soberana

2026 · github.com/HormigasAIS/LBH-Net · v1.5.0

Red mesh P2P completa construida desde cero. Los nodos se descubren automáticamente via broadcast UDP sin Git, sin registro central, sin nube.

  • Autodescubrimiento UDP (puerto 9199) — nodos se unen solos
  • Servidor TCP (puerto 9200) — transporte en tiempo real
  • Identidad criptográfica Ed25519 por nodo
  • Cola SQLite + bloqueo de workers concurrentes
  • Heartbeat + autocura + retry + backoff + dead-letter queue
  • v0.1 → v1.5 en una sola sesión de desarrollo
