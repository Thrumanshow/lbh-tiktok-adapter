lbh-tiktok-adapter — TikTok E-commerce Protocol Adapter

2026 · github.com/Thrumanshow/lbh-tiktok-adapter

Binary protocol adapter for TikTok E-commerce order processing. Transforms JSON order events into 16-byte LBH pheromones with CRC-16 integrity verification.

  • 89.3% bandwidth reduction — 150 bytes JSON → 16 bytes binary
  • At 1B events/day: 139 GB → 14.9 GB saved daily
  • Claim-based SQLite locking — zero race conditions across workers
  • Retry + exponential backoff + dead-letter queue
  • RFC-LBH-0004 (Data Integrity) + RFC-LBH-0006 (Signal Standard)
  • Architecture for Salesforce Commerce Cloud, Shopify, Channel Advisor

LBH-Net — Sovereign P2P Swarm Network

2026 · github.com/HormigasAIS/LBH-Net · v1.5.0

Full P2P mesh network built from scratch. Nodes discover each other automatically via UDP broadcast without Git, without central registry, without cloud.

  • UDP autodiscovery (port 9199) — nodes join automatically
  • TCP socket server (port 9200) — real-time signal transport
  • Ed25519 cryptographic identity per node
  • SQLite queue + concurrent worker locking
  • Heartbeat + autocure + retry + backoff + dead-letter queue
  • v0.1 → v1.5 in a single development session
